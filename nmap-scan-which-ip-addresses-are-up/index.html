Nmap: Scan Which Ip Addresses Are Up

Dec 22nd, 2010 12:00 am

Well, I write this article because of my pure ignorance. I’m quite into Network and Security right now.

I’m always reset my router and rearrange DHCP. And sometimes I cannot find ip addresses of my other computers and I don’t want to walk to those computers and ifconfig them individually. So, there must be other ways to know which ip addresses are up in the network.

These days, there are several port scanners over the Internet, but I’m old school. So, I go back to the most basic thing, nmap

You just install nmap, you can use apt-get install nmap or port install nmap in Linux and Mac. After that, just type this command.

1
nmap -sP 192.168.1.0/24

This command will scan the ip address from 192.168.1.0 – 192.168.1.255 and report which ip addresses are up.
