FixIt : Misdirected Shortcuts

Mar 19th, 2008 12:00 am

I have found this problem from my sister notebook. She had a problem which the registry was unintentionally cleaned. She run a registry program, and it did clean the registry, but it did more than that. So, my sister computer cannot find the shortcut file type which made all of shortcuts dead, and the shortcut file type was gone. For instance, when she clicked the programs on Start Menu; that programs are made from lnk file type. Those programs didn’t respond, because the computer cannot find shortcut file type. The problem can be solved by fixing shortcut file type.

  1. Follow http://filext.com/faq/broken_exe_association.php.

  2. 1.1 In create new extension, if you cannot find ShortCut file type please follow this solution http://www.dougknox.com.

  3. Go to Doug’s website and find Windows XP fixes -> File Association Fixes -> LNK (Shortcut) File Association Fix and run the registry file they provided.

  4. It should work just fine, at least my sister computer works.
