HowTo : Contact Hotmail <--> Microsoft Outlook Import and Export

Aug 20th, 2007 12:00 am

Import contacts from Windows Live Hotmail to Outlook{.OAnc}.

  1. Log on to your Windows Live Hotmail account.
  2. In the left pane, click Contacts.
  3. On the Windows Live Hotmail toolbar, click Options, then click Import Contacts.
  4. Under Option 1: Fast import for Outlook or Outlook Express, click Download and run Contacts Importer.
  5. When prompted to run or save the file, click Run.??Note??????If you use a Web browser other than Windows Internet Explorer, you must save the Windows Live Contact Importer to your computer before you can run it.
  6. When prompted to run the software, click Run.
  7. In the Windows Live Contact Importer wizard, click Next.
  8. Type your Windows Live ID (your e-mail address) and password, and then click Next.
  9. Select the check boxes for address books you want to import, and then click Next.
  10. All contacts are selected by default. Clear the check boxes for any contacts that you do not want to import, and then click Next.

Export contacts from Outlook to Windows Live Hotmail{.OAnc}.

  1. Log on to your Windows Live Hotmail account.
  2. In the left pane, click Contacts.
  3. On the Windows Live Hotmail toolbar, click Options, then click Export Contacts.
  4. Under Export contacts, click Export contacts.
  5. When prompted to open or save the file, click Save.
  6. Select a location to save the file, and then click Save.??Note??????The default file name is WLMContacts.csv.
  7. In Outlook, on the File menu, click Import and Export.
  8. Click Import from another program or file, and then click Next.
  9. Click Comma Separated Values (DOS), and then click Next.
  10. Click Browse to locate the file you saved in step 6, select the file, and then click OK.
  11. Under Options, choose an option for how to process duplicate contacts, and then click Next.
  12. Click the Outlook contacts folder where you want the imported contacts to be saved, and then click Next.??Note??????Many people only have one Contacts folder. However, if you have multiple contacts folders, you should pick the folder where you want your Windows Live Hotmail contacts to be saved.
  13. Click Finish.
