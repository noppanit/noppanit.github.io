Vi, Search and Replace String the Whole File

Apr 1st, 2011 12:00 am

Well this is actually nothing much, but when I was editing 1MB of sql dump file, mate doesn’t do a very good job because it’s very slow. So, I go for vi and use this simple command

1
:%s/<find_pattern_string>/<new_string>/g

And bam every string in the 1MB file changed in a second.
