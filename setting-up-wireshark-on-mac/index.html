Setting Up Wireshark on Mac

Dec 21st, 2010 12:00 am

I know this might sound stupid and somebody might say “Just double click the DMG file”. Well, I have so much troubles without reading the ReadMeFile.txt. And I think this website is really helpful if you guys are into tcpdump and want to learn more about Wireshark

How to install properly
http://josephhall.org/nqb2/index.php/2009/07/21/wrshrkinstll

And after you restart your mac, If you get this error “Insecure Startup Items folder detected.
Items in the Startup Items folder (“/Library/
StartupItems/”) have not been started because the
folder does not have the proper security settings”

Then do this, this is to set permission to /Library/StartupItems

1
2
sudo chown -R root:wheel /Library/StartupItems
sudo chmod -R 0755 /Library/StartupItems

Ref: http://reviews.cnet.com/8301-13727_7-10439851-263.html

And that’s it. You will see your interfaces come up in Wireshark now, if you don’t set permission properly, the app will give you an error.
